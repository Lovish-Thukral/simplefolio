LUNA – Privacy Focused Chat App for Couples

LUNA is a unique one-on-one chat application built with React Native and Node.js, designed exclusively for couples. It uses end-to-end encryption and a hash-key pairing system to ensure privacy without storing sensitive data on the server. The app includes custom logging mechanisms and focuses on security, exclusivity, and privacy-first communication.
